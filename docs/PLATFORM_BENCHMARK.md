# 平台调研备忘

## 主流平台启发

Koyfin：
- 强在自定义看板、图表、关注列表和组合分析。
- 对 Luvio 的启发：专业用户需要自己组织信息，而不是被固定页面牵着走。

TIKR：
- 强在筛选器、财务数据、估值倍数和关注列表动线。
- 对 Luvio 的启发：从“发现公司”到“进入研究”的路径必须短。

Quartr：
- 强在一手 IR 材料、电话会、公告和演示资料。
- 对 Luvio 的启发：AI 研究必须扎在可信来源上。

AlphaSense：
- 强在海量文档搜索、摘要和领域研究工作流。
- 对 Luvio 的启发：搜索、摘要和证据引用要合成一个动作。

Bloomberg Launchpad：
- 强在专业研究界面、监控组件和高频信息组织。
- 对 Luvio 的启发：高级用户需要可配置的研究空间。

## Luvio 的产品楔子

- 中文港股优先。
- 价值投资工作流优先。
- 先暴露证据缺口，再生成结论。
- 投资备忘录是一等对象。
- 组合页关注纪律、集中度和复盘，不制造交易冲动。

## 体验取舍

应该做：
- 首页直接进入研究室。
- 公司切换始终轻量。
- 首屏只保留研究助手、当前公司和下一步动作。
- 复杂信息收进标签页和档案区。
- 来源材料状态在报告前可见。

不应该做：
- 用满屏行情制造焦虑。
- 把所有模块同时摊开。
- 没有来源就给确定结论。
- 把备忘录当成报告导出附件。
